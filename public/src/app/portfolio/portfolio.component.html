Staff Clock In/Out System

A clock out system for employees and a view section for managers.Try it(Just example, Fake emails only)

  • The index Pg being the Dashboard, will display your clock in and clock out time with a date and allow you to clock out if you haven't already.

  • If an employee clocks out, a report of the day can be submitted.

  • In the manager view, the view will dislay those who have clock in for the day. I a report was submitted successfully, it can be viewed here.

  • The manage view section has the options to add an employee, edit an employees account and add points to that employee.

  • A logged in user can change a password if need be.
